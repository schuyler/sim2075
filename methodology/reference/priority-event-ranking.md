## Ranking Methodology

Events ranked by **annual probability** using pairwise comparison. When absolute estimates contradict this ranking, something is wrong—revise until coherent.

Ranking is judgment, not measurement. But relative comparisons are often more reliable than absolute estimates.

**Causal type** affects how probability is interpreted:
- **Type 1** (Stochastic): Probability is base rate; timing unpredictable
- **Type 2** (Threshold): Probability depends on current pressure level
- **Type 3** (Contingent): Probability = P(window) × P(resolution | window)

## Pairwise Comparison Notes

### Financial vs. Geopolitical

**Global Financial Crisis > Any single state failure**
- Financial crises are systemic and recurrent
- State failures require specific country conditions to align
- GFC: ~3%/year vs. Pakistan: ~1.5%/year ✓

### State Failures Relative (All Type 2)

**Pakistan ≈ Egypt > Saudi > Russia**
- Pakistan: Highest pressure (water + food + fiscal + nuclear)
- Egypt: Similar but slightly better external support (higher threshold)
- Saudi: Appears fragile but massive fiscal buffers (much higher threshold)
- Russia: Regime consolidated; pressure lower than others

### Climate Events Relative (All Type 2)

**AMOC ≈ Amazon > Permafrost**
- Both AMOC and Amazon approaching thresholds
- Permafrost is slower (feedback rather than tipping point)
- All are ~1% or below annually at current pressure

### Great Power Conflicts Relative (All Type 3)

**Taiwan ≈ India-Pakistan ≈ Korea**
- All have nuclear dimension creating deterrence
- All have persistent tension but active conflict avoidance
- Taiwan slightly higher due to explicit reunification timeline
- All require **window + resolution** → multiple pathways to non-conflict outcomes

## Using This Ranking

When specifying a new event:

1. **Determine causal type** — Is this Type 1, 2, or 3?
2. **Find neighbors** — Which events of the same type is yours most similar to?
3. **Make pairwise judgment** — Is your event more or less likely than each neighbor?
4. **Check coherence** — Does your absolute estimate match the implied ranking position?
5. **Revise if needed** — If estimates contradict ranking, something is wrong

**Example:**
> Specifying "Nigeria State Failure"
> - Causal type: Type 2 (threshold)—pressure accumulates from multiple sources
> - Similar to: Pakistan, Egypt (Type 2 state failures)
> - Pairwise: Nigeria probably less likely than Pakistan (less acute water crisis), similar to or slightly below Egypt
> - Implied range: 1-2% annual
> - If my estimate was 3%, that would contradict the ranking—I'd need to justify why Nigeria is more likely than Egypt

**Example:**
> Specifying "Major Climate Agreement"
> - Causal type: Type 3 (contingent)—requires negotiation window + successful resolution
> - Similar to: Taiwan (Type 3), but with different resolution distribution
> - Treaties have ~30-40% success rate when window opens (vs. conflict ~35% for Taiwan)
> - Estimate P(window) × P(success) for overall annual probability
