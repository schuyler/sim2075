# Planned Event Categories - Discontinuity Model

Reference document for the Monte Carlo simulation event catalog. These events were identified as priority candidates based on the methodology document, collapse-patterns framework, and existing research.

## 1. Climate and Planetary Tipping Points (5-8 events)

These are the slow-building but potentially irreversible discontinuities. The methodology document notes these have scientific literature providing probability ranges, making them good starting points for calibration.

| Event | Probability Range | Key Characteristics |
|-------|------------------|---------------------|
| **AMOC Significant Weakening** | 1-2.5%/year | Transforms European climate; effectively permanent |
| **Amazon Tipping Point** | 1-2%/year | Forest-to-savanna transition; global climate feedback |
| **Permafrost Methane Release** | 0.5-1.5%/year | Accelerates warming; irreversible |
| **Arctic Sea Ice Collapse** | 2-4%/year (ice-free summer) | Albedo feedback; opens shipping routes |
| **West Antarctic Ice Sheet Destabilization** | 0.5-1%/year | Commits to multi-meter sea level rise |
