# Source Registry Index

Evidence library supporting the simulation's parameters and worldview.

---

## Overview

This directory contains source notes documenting external evidence — papers, articles, reports, data — that inform event specifications and synthesis documents. Each source note captures metadata, key claims, quality assessment, and links to what it informs.

**Standards:** [[methodology/reference/research-documentation-standards]]

**Template:** [[methodology/05-source-template]]

## Directory Structure

```
sources/
  index.md              # This file
  articles/             # News, magazine, opinion pieces
  papers/               # Academic/peer-reviewed
  reports/              # Think tanks, IGOs, NGOs, government
  data/                 # Datasets, statistical releases
  books/                # Book chapters, excerpts
```
